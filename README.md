# 🎓 AI Mentor: Personalized Learning Assistant

[![Hugging Face Space](https://img.shields.io/badge/%F0%9F%A4%97%20Hugging%20Face-Space-blue)](https://huggingface.co/spaces/AlhaJibran/AI_Mentor)
[![Python 3.10+](https://img.shields.io/badge/python-3.10+-blue.svg)](https://www.python.org/downloads/)

AI Mentor is a generative AI application designed to help learners of all levels master any topic. By leveraging high-speed inference and state-of-the-art LLMs, the app provides tailored educational content ranging from basic theory to complex industry practices.

---

## 🌟 Features

- **Adaptive Learning Levels:** Choose between Beginner, Intermediate, and Advanced paths.
- **Customizable Modules:** Select specific learning modes including Theory, Problems, Industry Practices, and Quizzes.
- **Lightning Fast:** Powered by **Groq LPU** for near-instant responses.
- **User-Friendly UI:** Built with Gradio for a clean web interface.

---

## 🛠️ Tech Stack

- **Language:** Python
- **LLM:** Llama 3.3 (via Groq Cloud)
- **UI Framework:** Gradio
- **Deployment:** Hugging Face Spaces
